Local install with ISO/RPM

Date: 2023/11/29
Auth: Guanyan.Wang
Environment: Redhat 8.1

1. Write repo

First:

 We create an repo called "local.repo" at "/etc/yum.repos.d" 

$ cd /etc/yum.repos.d
$ touch local.repo

Second:

 We write below content in "local.repo" 

[BaseOS-local]
name=BaseOS-local
enabled=1
baseurl=file:///mnt/cdrom/BaseOS/
gpgcheck=0
[AppStream-local]
name=AppStream-local
baseurl=file:///mnt/cdrom/AppStream/
gpgcheck=0

Third:

 mount the ISO file to specific path, in this case, we specify to "/mnt/cdrom" 
$ mount /path/to/ISO/File /mnt/cdrom

2. Update your yum/dnf repolist

Just simply run command:

$ dnf repolist
$ yum repolist

Now, you can install package from local host !
